# RideBuddy Carbon Tracker

## Description
The **RideBuddy Carbon Tracker** calculates and visualizes carbon emissions for various transportation modes. Users can input trip details and analyze their carbon footprint through interactive charts.

## Features
- Customizable input fields for trip details.
- Dynamic emission calculations for cars and bikes.
- Graphical visualization of emission history using Chart.js.
- Responsive design optimized for all devices.

## Technologies Used
- HTML5, CSS3, JavaScript
- Chart.js for data visualization
- Google Fonts (Montserrat)
